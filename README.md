# תרומה למאגר השאלות (Q&A)

תודה על הרצון לתרום למאגר!

## איך מוסיפים שאלות?

### 1. שכפול הפרויקט

יש לבצע אחד מהבאים:

* Fork למאגר האישי שלכם ולאחר מכן Clone למחשב המקומי.
* Clone ישירות למאגר (למי שיש הרשאות כתיבה).

### 2. הוספת תוכן

בכל מודול קיימים:

* קובץ `questions.md` המכיל את כל השאלות.
* תיקיית `answers/` המכילה את התשובות.

בעת הוספת תוכן חדש:

1. הוסיפו את השאלה לקובץ `questions.md`.
2. צרו קובץ תשובה מתאים בתוך תיקיית `answers/`.
3. הקפידו על שמות ברורים ועקביים בין השאלה לתשובה.

דוגמה:

```text
<module>/
├── questions.md
└── answers/
    └── <answer-file>.md
```

### 3. פורמט השאלות

בקובץ `questions.md`:

```markdown
## מה ההבדל בין Process ל-Thread?

ראו תשובה בתיקיית answers.
```

### 4. פורמט התשובות

בקובץ המתאים תחת `answers/`:

```markdown
# מה ההבדל בין Process ל-Thread?

תוכן התשובה...
```

### 5. יצירת Commit

לאחר סיום העריכה:

```bash
git add .
git commit -m "הוספת שאלות ותשובות למודול <שם המודול>"
```

### 6. פתיחת Pull Request

* בצעו Push לשינויים שלכם.
* פתחו Pull Request מול המאגר הראשי.
* תארו בקצרה אילו שאלות ותשובות נוספו או עודכנו.

## כללי תרומה

* עברית בלבד.
* שאלות מקצועיות ורלוונטיות לנושא המודול.
* יש להימנע משאלות כפולות.
* תשובות צריכות להיות מדויקות, ברורות ומעודכנות.
* יש לשמור על מבנה התיקיות הקיים.
* כל שאלה חדשה חייבת לכלול תשובה מתאימה.

תודה על התרומה והעזרה בשיפור המאגר!
